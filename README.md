# YouTube Thumbnail Showcase Website

A modern, responsive portfolio website showcasing professionally designed YouTube thumbnails. Built with HTML, CSS, and featuring a dark theme with an elegant profile section and thumbnail gallery grid.

## 🎨 Features

- **Professional Profile Section** - Displays creator information with profile picture, bio, and key statistics
- **Responsive Thumbnail Gallery** - Grid-based layout showcasing 9+ thumbnail designs
- **Dark Theme Design** - Modern dark interface with orange accent colors for better visual appeal
- **Social Media Integration** - Quick links to GitHub and LinkedIn profiles
- **Mobile Responsive** - Fully optimized for desktop, tablet, and mobile devices
- **Modern CSS Styling** - Custom CSS with CSS variables for easy theme customization

## 📁 Project Structure

```
YouTube-Thumbnail-Website/
├── index.html          # Main HTML file with profile and gallery structure
├── index.css           # Styling with dark theme and responsive design
├── README.md           # Project documentation
└── [images]/           # Thumbnail images (1.jpg, 2.jpg, 3.jpg, etc.)
```

## 🚀 Quick Start

1. **Clone or download the project**
   ```bash
   git clone https://github.com/WajidKorkani/YouTube-Thumbnail-Website.git
   cd YouTube-Thumbnail-Website
   ```

2. **Add your thumbnail images**
   - Place your thumbnail images (1.jpg, 2.jpg, 3.jpg, etc.) in the project root directory
   - Update image sources in `index.html` if using different filenames

3. **Open in browser**
   - Simply open `index.html` in your web browser
   - No build tools or server setup required!

## 🎯 Customization

### Modify Profile Information

Edit the profile section in `index.html`:
- Update the profile picture URL
- Change name and bio text
- Modify statistics (50+ Thumbnails, 3+ Years Experience)
- Update social media links (GitHub, LinkedIn)

### Update Thumbnail Gallery

Add or remove thumbnail items in `index.html`:
```html
<div class="thumbnail-item"><img src="your-image.jpg" alt="Your thumbnail"></div>
```

### Customize Colors

Modify CSS variables in `index.css`:
```css
:root {
    --bg-color: #0f0f0f;           /* Background color */
    --accent-color: orange;         /* Accent color (currently orange) */
    --text-main: #ffffff;           /* Main text color */
    --text-dim: #aaaaaa;            /* Dim text color */
}
```
